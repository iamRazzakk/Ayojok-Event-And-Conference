```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The ans is "12". In the call sum(1, "2"), the second argument "2" is a string, not a number. JavaScript is a dynamically typed language, which means it will try to perform type coercion to convert the string to a number when using the + operator.</i>

</p>

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The code counts and logs the number of truthy values in the nums array. In JavaScript, numbers other than 0 are truthy. Since the array contains [0, 1, 2, 3], which includes three truthy values (1, 2, and 3), the output is 3.</i>
